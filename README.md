# 運命の舟券 〜Luck Pick〜

## バージョン

β v0.9.0

## 概要

各艇への期待度をゲージで設定し、1000回の重み付きシミュレーションから3連単ランキングを表示するブラウザアプリです。
ランキング生成後は、全シミュレーション結果の出現回数を重みとして「運命の1点」を抽選できます。

## 使い方

1. 来そうだと思う艇ほどゲージを右へ動かします。
2. 「1000回シミュレーション」を押します。
3. 上位10通りのランキングを確認します。
4. 「運命の1点を決める」を押して、買い目を1つ抽選します。
5. 設定を戻す場合は「リセット」を押します。

## 公開ファイル

- `luck-pick-beta-v0.9.0.html`

現時点ではHTML単体で動作します。サーバー処理やデータベースは不要です。

## 動作上の特徴

- 1000回の抽選処理はWeb Workerで実行
- 1着、2着、3着を順番に重み付き抽選
- 同一艇の重複なし
- ランキング上位10通りを表示
- 「運命の1点」は全1000件の結果から抽選
- PC・スマートフォン対応
- ゲージ数値は非表示
- 艇番ごとにつまみの色を変更
